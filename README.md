# 🛍️ UrbanStyle - E-Commerce Platform

UrbanStyle is a modern e-commerce web application built using React and Vite. It offers a clean and minimal streetwear-themed UI with essential shopping features like product browsing, cart management, and wishlist functionality.

## 🚀 Features
- 🛒 Add to Cart with size selection  
- ❤️ Wishlist management  
- 🔍 Product detail pages  
- 🔄 Dynamic routing using React Router  
- ⚡ Fast performance with Vite  
- 🎨 Responsive UI with Tailwind CSS  

## 🛠️ Tech Stack
- React (Hooks + Context API)  
- Vite  
- Tailwind CSS + ShadCN UI  
- React Router DOM
